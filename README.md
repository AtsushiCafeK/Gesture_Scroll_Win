# Gesture Scroll Win

## 概要
このアプリはWindows用です。
Panasonicなど一部のメーカーでトラックパッド上で円を描くとスクロールできる機能がありますが、ソフトウェアで類似した操作を可能にするものです。一般的な四角いトラックパッド上で実行することができます。
まだ使用者サンプルが少ない為、調整が粗い状態です。

## 初回設定
1. ダウンロード後、zipファイルを解凍
2. アプリをスタートアップへ登録
3. トラックパッドのジェスチャーへアプリの有効無効の操作を登録
4. アプリを起動して有効状態にしてスクロールが実行可能になる

## 使い方
1. アプリ起動
2. ON/OFF切替 で有効(ON)にする。有効になるとタスクバーのアイコンが赤色になる、オフは青色になる。
3. カーソルを円状に回転するとスクロールする。

## 機能
- 回転方向の反転
- ホットキー変更
- スクロール速度変更
- 起動時にGUIを隠す。トレイ常駐
- スタートアップへショートカット作成

## 補足
